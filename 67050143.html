My Project

Smart Parking System

Smart Parking System is an IoT-based parking management project designed to monitor parking space availability in real time. It uses an ESP32 microcontroller integrated with ultrasonic sensors to detect vehicle presence in each parking slot. Real-time data is visualized on an OLED display and a web dashboard, enhancing parking efficiency and demonstrating smart city applications.
